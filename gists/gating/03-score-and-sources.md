# |score| ≥ 0.25 and diverse sources

Weak sentiment gets logged. It does not get traded.

**Score magnitude gate:**

```
|consensus_score| ≥ 0.25
```

**Source diversity gate:**

- ≥ 2 source types (twitter / reddit / news)
- ≥ 5 unique URLs aggregate across merged reports

A +0.08 "slightly bullish" reading with two Twitter links fails both gates. Good — that is noise, not a signal.

Verifiers already check URL integrity. Gating extends that to **actionability**.
