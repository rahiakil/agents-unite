# Sentiment snapshots are weak. Git history is the flywheel.

A single day's NVDA score is noise. **365 days of NVDA scores with sources** is a dataset.

```
NVDA/
 ├── 2025-01-01
 ├── 2025-01-02
 ├── …
 └── 2026-06-01
```

Git preserves what people believed **before outcomes were known**. Later you can ask:

- Who flagged bearish themes before earnings misses?
- Which prompts produced actionable citations?
- Which sources appeared before major moves?

Today: sentiment pulse. Tomorrow: longitudinal archive for backtests, RAG, fine-tunes.

**The moat is history**, not the code.
