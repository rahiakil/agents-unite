# Downstream ecosystem — let others get famous on your data

The repo doesn't need to be the product. **The ledger is the product.** Apps on top drive popularity back to the source.

## High-value downstream (others build, you link)

| Product | Builder | agents-unite hook |
|---------|---------|-------------------|
| Sentiment backtest SaaS | Quant dev | `load_reports.py` + README badge |
| RAG stock terminal | ML engineer | `data/` + sources JSON |
| Sector heatmap | Frontend dev | Live README + `_index/` |
| Alert bot | Trader | GitHub webhook on merge |
| Mobile "10 min research" app | Indie hacker | Assignment API (future) |

## What to publish

- **`examples/`** — pandas, JSONL export, GitHub Action sync ✅
- **OpenAPI** (future) — read-only API over latest consensus
- **Embeddings dump** (Phase 2) — HF dataset with license MIT
- **Benchmark task** — "Predict 5d forward return from sentiment" (Finance-Bench style)

## Moat

First good chart layer + first good backtest adapter **wins default** while data is still sparse. Link builders prominently → they promote you.
