# Reputation as marketing — Phase 4 is your growth engine

Trust and popularity merge when **names mean something**.

## Why reputation drives growth

- Contributors stay for **scoreboard**
- Readers trust **weighted consensus** over anonymous sentiment
- Funds pay for **verified** alt-data, not raw scrapes

## Phase 4 features that market themselves

| Feature | Public face |
|---------|-------------|
| Contributor ELO | README leaderboard |
| Prediction tracking | "Called NVDA move 3 days early" badges |
| Verifier accuracy | "Audited by @user (98% URL validity)" |
| Stake-gated signals | Premium tier for builders, free data remains |

## Interim (before Phase 4)

- Show **github_username** on every report ✅
- Track **merged PR count** per user in live README
- **Verifier thank-you** line in consensus.md footer

## Research angle

Independent measurements + reputation = **ensemble you can trust** — publishable claim for academic/partner outreach.
